
Following Versions are used for the project:
	NodeJS 	: v8.11.1
	MongoDB	: 3.6
	Angular : 6.0
	Express	: 4.16.3 


All the services used for the functionality is placed in 
	posts.service.ts
	
Server path : sample-tacto-project\src\Server

api are placed in path : sample-tacto-project\src\Server\routes

Please follow below steps to run the project

==>go to path : sample-tacto-project

	Step 1 : Run "npm install" to install the dependencies

	Step 2 : Run Mongodb on your system

==>go to path : sample-tacto-project\src\Server

	Step 3 : Run "node server.js" to install the dependencies

==>This Step is for Data Setup in MongoDB

	Step 4 : go to URL : "http://localhost:3000/api/setupdata"
	
	Step 5 : http://localhost:3000
